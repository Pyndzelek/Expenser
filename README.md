# Expenser

#### This repo was initiated with template dedicated for this tech stack: https://github.com/Pyndzelek/Monorepo-Nextjs-Bun-template

## Expenser is a project created simply to practice creating API with Hono

## Tech stack

- **Next.js** - React framework for production
- **Bun** - Fast all-in-one JavaScript runtime
- **Hono** - Lightweight web framework for building APIs
- **Drizzle ORM** - TypeScript ORM for type-safe database queries
- **Zod** - TypeScript-first schema validation
- **PostgreSQL** - Relational database
